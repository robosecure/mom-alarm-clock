3-Tier Architecture: What Works at Each Level

Tier 1: App Store Only

  • Full alarm state machine (idle, active, verified, locked)
  • Escalation ladder with timed threshold crossings
  • Four+ wake verification methods
  • Parent push notifications at each escalation step
  • Critical Alerts bypass DND and silent mode
  • Bidirectional parent-child messaging
  • Streak tracking, on-time stats, history log
  • Persistent state via iCloud sync

Tier 2: App + Screen Time / FamilyControls

  • Morning Lock via FamilyControls (block all non-allowed apps)
  • DeviceActivityMonitor for app-usage enforcement
  • Volume floor enforcement via AVAudioSession
  • Prevent alarm app deletion (Screen Time restriction)
  • Background audio session to survive app switch
  • Heartbeat monitoring: detect force-quit within minutes
  • Requires Family Sharing with child Apple ID

Tier 3: App + MDM (Maximum Enforcement)

  • MDM profile prevents app uninstall without parent PIN
  • Restrict access to Settings during alarm window
  • Prevent notification permission changes
  • Kiosk-mode option: lock device to alarm app
  • Remote wipe / lock if device is lost
  • Enterprise-grade tamper detection
  • Requires MDM enrollment (Jamf, Mosyle, etc.)
